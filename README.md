# 🧠 Cyberpunk AI Portfolio.

> **"WAKE UP, SAMURAI. DATA NEVER SLEEPS."**

A futuristic **AI / Data Science themed portfolio website** built with **React, TypeScript, and Tailwind CSS**.
This portfolio showcases **machine learning projects, AI tools, and data science work** in a cyberpunk hacker-style interface.

---

# 🚀 Features

## 🎨 Cyberpunk UI

* Terminal-style interface with hacker aesthetics
* Glitch text animations
* Neon cyberpunk color scheme
* Matrix rain background animation
* Responsive modern design

## 🧠 AI & Data Science Projects

The portfolio highlights real machine learning projects including:

* **Text-to-Speech Generator** – Speech synthesis using HuggingFace Transformers
* **AI Scholarship Predictor** – ML model predicting scholarship eligibility
* **AI Study Planner** – Intelligent system for planning study schedules
* **Weather Forecasting System** – Machine learning weather prediction
* **News Classifier** – NLP model for news categorization
* **Movie Recommendation System** – Recommendation engine using SVD
* **News Sentiment Analyzer** – NLP sentiment analysis on news data

# ⚙️ Setup & Installation

## 1️⃣ Clone the repository

```
git clone https://github.com/govindchauhan45/portfolio.git
cd portfolio
```

## 2️⃣ Install dependencies

```
npm install
```

## 3️⃣ Run the development server

```
npm run dev
```

## 4️⃣ Build for production

```
npm run build
```
